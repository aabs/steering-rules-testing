:::rule id="AI-001" mandatory="true" category="agentic" 
Never leave temporary files in the repo after they are no longer in use.
:::

:::rule id="AI-002" mandatory="true" category="agentic" 
When developing diagnostic, experimental or temporary test code, make sure it is created and run somewhere other than the source repo.  This prevents dead code being left lying around.
:::
